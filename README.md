FnP Sales Analysis Dashboard

End-to-end analytics project transforming raw sales data into interactive executive insights using Excel's native Power tools.

Project Overview:

Complete 6-step BI pipeline analyzing 1,000 FnP orders (₹3.5M revenue):

    Data Sources: Order DB, Customer DB, Delivery tracking

    Key Metrics: ₹3,521 AOV, 5.53-day avg delivery

    Analysis: Revenue by occasion/category, top cities, hourly trends

Technical Implementation:

Power Query ETL (Steps 1-3) 
    ↓
Power Pivot Modeling (Step 4) 
    ↓
Pivot Tables + DAX Analysis (Step 5)
    ↓
Interactive Excel Dashboard (Step 6)

Features:

    25+ transformed columns (delivery days, hour buckets, customer segments)

    Multi-table dimensional model with 5 relationships

    Interactive slicers (date, occasion filters)

    7 core visualizations (revenue trends, top cities, hourly patterns)

Key Insights Delivered:

    Peak occasions: Anniversary, Birthday, Diwali

    Top categories: Cakes, Combos, Flowers (75%+ revenue)

    Geographic concentration: Top 5 cities = 50% orders

    Evening ordering surge (6-10 PM)

Tech Stack:

Power Query (ETL) | Power Pivot (Modeling) | DAX (Measures) | Excel (Dashboard)
